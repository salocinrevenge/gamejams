This is a compilation of visual and sound effects to be used in game and animation projects.

- The tank-king folder contains effects copied from github [https://github.com/tank-king/Tutorials.git](https://github.com/tank-king/Tutorials.git). Go check it out!

- The pygame-grass-main folder contains a grass rendering and physics system for pygame. check [https://github.com/DaFluffyPotato/pygame-grass.git](https://github.com/DaFluffyPotato/pygame-grass.git) for more info.
